# Employee CRUD Application (Vue.js)

## Project Description
This is a simple Employee Management System built using Vue.js, Bootstrap, Axios, and MockAPI.  
It demonstrates full CRUD operations:
- Create Employee
- Read Employee
- Update Employee
- Delete Employee

---

##  Features
- Add new employee
- View employee list
- Update employee details
- Delete employee records
- Responsive UI using Bootstrap

---

## Tech Stack
- Vue.js
- Bootstrap 5
- Axios
- MockAPI (REST API)

---
